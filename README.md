## 📖 Overview

This repository contains the source code for the **QuizNest Brain** official landing page. It serves as the central hub for users to learn about the app's features, understand the BDApps subscription model, and download the official Android APK.

The website is designed with a modern **glassmorphism UI**, fully responsive layout, and a seamless **dark/light mode** toggle, ensuring a premium user experience across all devices.

## ✨ Key Features

- **Modern UI/UX:** Built with vanilla HTML/CSS utilizing CSS variables for easy theming.
- **Dark/Light Mode:** Integrated theme toggler that respects user preferences and saves them via `localStorage`.
- **Responsive Design:** Fluid typography and flexbox/grid layouts that adapt perfectly from mobile screens to large desktop monitors.
- **App Showcase:** Clearly communicates the app's core value propositions:
  - Passwordless BDApps OTP Login
  - Robi and Airtel Subscription Integration
  - Diverse quiz categories and detailed analytics
- **Direct APK Download:** Secure and direct link to download the latest Android application release.

## 🚀 Getting Started

Because this is a static website, no build step or package installation is required. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rhsohan/quiznest_brain_website.git
   ```
2. **Open the project:**
   Simply open the `index.html` file in any modern web browser to view the site.
3. **Deploy:**
   The site can be deployed instantly via GitHub Pages, Vercel, Netlify, or any standard web hosting provider.

## 🎨 Technologies Used

- **HTML5:** Semantic structuring and accessible layout.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, animations, and glassmorphism (backdrop-filter).
- **Vanilla JavaScript:** DOM manipulation for theme toggling.
- **Google Fonts:** Utilizing *Outfit* and *Plus Jakarta Sans* for premium typography.
- **FontAwesome (v6.4.0):** Scalable vector icons.

## 📱 About The App

QuizNest Brain is exclusively designed for Android 8.0+ users in Bangladesh on the **Robi** and **Airtel** networks. It allows users to:
- Subscribe easily via their mobile balance.
- Login instantly using BDApps OTP without remembering complex passwords.
- Test their knowledge across various curated categories.
